**Remote Over Radio — умный дом через LoRa, когда интернет недоступен или заблокирован**

Живёшь в доме на колёсах? На яхте? На даче, где сигнал пропадает в самый неподходящий момент? Rover решает эту проблему радикально — выбрасывает интернет из цепочки управления вообще.

Телефон → Bluetooth → LoRa-радио → километры воздуха → шлюз → Home Assistant. Никаких облаков, никаких серверов, никаких подписок. Работает даже когда оператор заблокировал всё, что можно.

## Как это устроено

Берёшь два LoRa-устройства на чипе Meshtastic. Одно вешаешь рядом с сервером HA, второе держишь при себе. Между ними — радиоканал дальностью до нескольких километров. Через него летят компактные команды в бинарном формате — включить свет, выставить температуру, проверить датчики.

## Система не сдаётся

Главная фишка Rover — **настойчивая доставка команд**. Отправил команду — она будет повторена, пока не дойдёт:

- Команды копятся в очереди и повторяются с настраиваемым периодом (по умолчанию каждые 15 секунд).
- Если не помещаются в один радиопакет — автоматически фрагментируются и собираются на приёмнике.
- При нескольких подключённых телефонах команды и обновления состояния доходят до всех.
- Активные алерты (открытая дверь, газ, дым) повторяются каждые 15 секунд, пока не сняты.
- Когда телефон возвращается на связь после паузы — он сам запросит актуальные состояния всех устройств.

Нажал «выключить генератор» и ушёл — система сама разберётся и доставит команду.

## Что умеет

- **Свет** — вкл/выкл, яркость, цветовая температура
- **Климат** — кондиционер, отопитель, бойлер с установкой температуры
- **Датчики** — температура, влажность, протечка, движение, вибрация, окна
- **Безопасность** — замки, сигнализация, сирены
- **Автоматика** — сцены, кнопки, любые переключатели
- **14 типов устройств HA** из коробки

## Железо

- **Шлюз**: Heltec Mesh Node T114 → USB → Home Assistant
- **Клиент**: SenseCAP Card Tracker T1000-E → Bluetooth → телефон
- Подойдёт любое устройство с прошивкой Meshtastic

## Установка

### Плагин Home Assistant (`back/`)
1. Добавить репозиторий в HACS: `https://github.com/BotoVed/Rover`
2. Установить Rover через HACS
3. Перезапустить Home Assistant
4. Settings → Integrations → Add → Rover
5. Выбрать порт шлюза, настроить канал и устройства

### Карточка Lovelace (`card/`)
1. HACS → Frontend → добавить репозиторий → выбрать Rover Card
2. На дашборде: Add Card → Custom: Rover Card
3. Карточка покажет статус системы, подключённые телефоны и поток пакетов

### Приложение (`app/`)
APK сборки — в [Releases](https://github.com/BotoVed/Rover/releases).
Или собрать самостоятельно — см. [`app/README.md`](./app/README.md).
