# Rover — app

Android-приложение для удалённого управления Home Assistant через LoRa.

## Что делает

- Подключается по Bluetooth LE к Meshtastic-устройству на телефоне.
- Принимает конфигурацию (зоны, устройства, маппинг) от шлюза.
- Отображает устройства HA с группировкой по зонам.
- Отправляет команды и принимает обновления состояний через LoRa.

## Установка

### Готовый APK
Скачать из последнего [app-релиза](https://github.com/BotoVed/Rover/releases?q=app), установить на Android (потребуется разрешение установки из неизвестных источников).

### Сборка из исходников
```bash
cd app/
export ANDROID_HOME=~/android-sdk
/tmp/gradle-8.6/bin/gradle assembleDebug --no-daemon
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

## Требования

- Android 8.0+ (API 26).
- Bluetooth.
- Meshtastic-устройство, спаренное с телефоном (например, SenseCAP T1000-E).
- Поднятая Rover-сеть с настроенным шлюзом.

## Первый запуск

1. Запустить приложение.
2. Перейти в Settings, ввести параметры подключения (имя канала, PSK, node ID шлюза). На MVP — вручную, в будущем — импорт через QR.
3. Просканировать BLE-устройства, выбрать Meshtastic-ноду.
4. Дождаться handshake, после `Ready` — запросить конфиг.
5. На главном экране появятся устройства с группировкой по зонам.
